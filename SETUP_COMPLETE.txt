QUICK START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Setup (One-Time):
   $ ./.github/setup-vnc-secrets.sh
   
   Requirements:
   - ngrok account (free): https://ngrok.com
   - ngrok auth token: https://dashboard.ngrok.com/auth

2. Run Workflow:
   GitHub → Actions → "VNC GUI Testing" → Run workflow

3. Connect:
   - Get VNC URL from workflow logs
   - Connect via VNC client or web browser
   - Start testing!
